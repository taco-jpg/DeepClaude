# DeepClaude

A seamless userscript wrapper that transforms DeepSeek's interface into a Claude-like experience.

## How to Use

### Prerequisites
- A modern web browser (Chrome, Firefox, Edge, etc.)
- A userscript manager extension installed (e.g., Tampermonkey, Violentmonkey, Greasemonkey)

### Installation Steps

1. **Install a userscript manager**  
   If you haven't already, add one from your browser's extension store.

2. **Add the script**  
   Simply drag and drop the `deepseek-model-router.user.js` file into any tab of your userscript manager extension.

3. **Reload the page**  
   Refresh the target website to activate the script.

That's it! You're all set to enjoy the enhanced interface.

---

### Note
Make sure your userscript manager is enabled and the script is active. If you encounter issues, try reinstalling or checking the extension's console for errors.
